# Getting started

`git clone https://github.com/chaithanya99/SPEcodeeditor.git`

`cd SPEcodeeditor`

## Setup Backend
- start mongodb

`cd backend`

`npm install`

`npm start`

## Setup Coderunner
`cd coderunner`

`npm install`

`npm start`

## Setup Frontend
`cd frontend`

`npm install`

`npm start`
